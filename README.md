# Portfolio
AP CSP Python Code
This is my portfolio consisting of projects I've created in my AP Computer Science Principles class. These projects range from interactive simulations to interactive, productivity tools useful for everyday life.
---
### Project 1: Hogwarts House Personality Generator

* **Project Title:** Hogwarts House Personality Generator
* **Summary:** This project is an interactive quiz that determines which Hogwarts house a user belongs to based on their personality traits. By answering a series of targeted questions, the program processes user input to calculate and output their ideal wizarding match.
* **Key Features:** Uses conditional logic (if-elif-else statements) to track scores for each house and handles input validation to ensure a smooth user experience.

### Project 2: Pet-Based Image Generator

* **Project Title:** Pet-Based Image Generator
* **Summary:** This application assists future pet owners by displaying a visual preview of the animals they are planning to adopt. Users select or input their desired animals, and the program dynamically retrieves and renders the corresponding imagery.
* **Key Features:** Integrates image-rendering assets or GUI elements based on user selections, demonstrating how code can manipulate and display visual media.

### Project 3: Turtle vs. Hare Race Game

* **Project Title:** Turtle vs. Hare Race Game
* **Summary:** This is a classic simulation game that brings the famous fable to life through a digital race between a turtle and a hare. The program automates movement and randomly determines the pace of each competitor until one crosses the finish line.
* **Key Features:** Utilizes loops, randomization algorithms, and basic computer graphics (such as the Python Turtle library) to create real-time movement and visual tracking.

### Project 4: Rollercoaster Finder

* **Project Title:** Rollercoaster Finder
* **Summary:** This utility application helps theme park enthusiasts find their next thrill by filtering a database of rollercoasters. Users can narrow down their search based on specific criteria like location, minimum height requirements, or intensity level.
* **Key Features:** Implements list/array traversal and data filtering techniques to parse through a collection of information and return customized results.

### Project 5: Slot Machine Game

* **Project Title:** Virtual Slot Machine Game
* **Summary:** This project simulates a classic casino slot machine experience right in the console or window. Users can place virtual bets, spin the reels, and watch the program evaluate the results to see if they won or lost.
* **Key Features:** Heavy reliance on random number generation, complex nested conditional statements to check for matching combinations, and a persistent variable to track the player's total wallet balance.

### Project 6: Grocery To-Do List Manager

* **Project Title:** Grocery To-Do List Manager
* **Summary:** This productivity tool allows users to digitally create, manage, and update a grocery shopping list. It provides a clean interface for keeping track of needed household items before heading to the store.
* **Key Features:** Showcases fundamental data structures by using lists to implement full CRUD (Create, Read, Update, Delete) functionality, allowing users to dynamically append or remove items.
